
# 🏥 MediSlot – Doctor Appointment Booking System

MediSlot is a full-stack healthcare web application designed to simplify the process of booking and managing doctor appointments. It provides separate dashboards for patients, doctors, and administrators, ensuring a smooth and efficient experience for all users.

---

## 🚀 Features

### 👤 Patient

* Register and login securely
* Browse doctors by specialization
* Book appointments بسهولة
* View and manage bookings
* Online payment integration

### 🩺 Doctor

* Secure login
* View assigned appointments
* Accept or reject bookings
* Mark appointments as completed

### 🛠️ Admin

* Add/remove doctors
* Manage all appointments
* Monitor platform activity

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/medislot.git
cd medislot
```

### 2️⃣ Install dependencies

**Frontend**

```
cd frontend
npm install
npm run dev
```

**Backend**

```
cd backend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in backend folder and add:

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
RAZORPAY_KEY_ID=your_key
RAZORPAY_SECRET=your_secret
```

## 💳 Payment Integration
